# Convert angular degrees to decimal degree values

Converts angular degree values to decimal degrees. Used in decimal
degree limit calculations.

## Usage

``` r
deg_to_dd(x)
```

## Arguments

- x:

  numeric to be converted

## Value

A vector of decimal degrees
