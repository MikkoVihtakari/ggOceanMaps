# Convert decimal degree values to angular degrees

Converts decimal degree values to angular degrees. Used in decimal
degree limit calculations.

## Usage

``` r
dd_to_deg(x)
```

## Arguments

- x:

  numeric to be converted

## Value

A vector of angular degrees
